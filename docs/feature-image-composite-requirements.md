# 溶图工具 — 开发需求文档

> 版本: v1.0 | 日期: 2026-05-30 | 状态: 待审核

---

## 一、功能概述

新增第 4 个工具「**溶图**」，支持用户上传一张底图 + 多张透明产品主图，通过拖拽交互将产品主图放置到画布任意位置，再通过「批量套模板」功能将多张产品主图快速排列到底图上，最终预览并逐张或批量下载合成结果。

**核心场景**：电商运营需要将产品主图（透明 PNG）快速合成到统一的营销底图上，生成多张商品展示图。

---

## 二、技术选型

| 层面 | 方案 | 说明 |
|------|------|------|
| 前端框架 | React 18 + Babel（CDN） | 与 image-tool / bg-remover 一致 |
| 样式 | Tailwind CSS（CDN） | 与现有工具保持统一 |
| Canvas 渲染 | 原生 Canvas API | 浏览端合成，无需后端参与 |
| 下载 | JSZip + FileSaver.js | 批量 ZIP 下载复用已有库 |
| 后端 | 无需新增 API | 纯前端实现，合成在浏览器完成 |
| 拖拽 | 原生 Pointer Events | 轻量、无额外依赖 |

## 五、功能模块详细设计

### 5.1 底图上传区

- **上传方式**：点击上传 / 拖拽上传 / Ctrl+V 粘贴
- **数量**：仅支持 1 张
- **格式**：JPG / PNG / WebP
- **替换**：已有底图时，点击新图替换（弹确认框）
- **删除**：底图旁显示 ✕ 按钮，移除后画布清空
- **尺寸限制**：不限制，自动读取原始分辨率用于 Canvas 渲染
- **缩略图**：侧边栏显示底图缩略图（最大 80px 高）

### 5.2 产品主图上传区

- **上传方式**：点击上传 / 拖拽上传 / Ctrl+V 粘贴（支持多选）
- **数量**：建议最多 20 张，超出提示
- **格式**：PNG（推荐透明背景）、JPG、WebP
- **列表展示**：缩略图 + 文件名 + 尺寸信息 + 选中高亮
- **选中交互**：单击选中，选中的图片在画布上显示蓝色边框和拖拽手柄
- **删除**：每张图右侧 ✕ 按钮，支持「清除全部」
- **排序**：支持拖拽调整列表顺序（影响模板排列顺序）

### 5.3 画布交互（核心）

画布使用 HTML Canvas 渲染，覆盖交互层处理拖拽事件。

#### 5.3.1 渲染层级
```
Layer 0: 底图（全尺寸渲染到 Canvas）
Layer 1: 产品主图（按用户放置位置绘制）
Layer 2: 选中框（蓝色虚线边框 + 缩放角点 + 旋转手柄）
```

#### 5.3.2 拖拽操作
| 操作 | 手势 | 说明 |
|------|------|------|
| 移动 | 拖拽选中的产品图 | 在底图上自由移动 |
| 缩放 | 拖拽角点手柄 | 等比缩放，按住 Shift 固定比例 |
| 旋转 | 拖拽顶部旋转手柄 | 15° 步进吸附，按住 Shift 自由旋转 |
| 删除 | 选中后按 Delete/Backspace | 从画布移除该产品图 |
| 复制 | Ctrl+D | 复制选中的产品图及其位置参数 |

#### 5.3.3 画布控制
- **缩放**：底部滑块控制画布显示缩放（25% ~ 400%），不改变实际合成尺寸
- **适应窗口**：双击缩放滑块自动适配画布区域
- **实际尺寸**：切换到 100% 显示实际像素大小
- **网格辅助线**：可选开启，显示三分线、中心线（辅助对齐）

### 5.4 批量套模板

模板功能将多张产品主图按照预设布局快速排列到底图上。

#### 5.4.1 内置模板

| 模板名 | 布局说明 | 适用场景 |
|--------|----------|----------|
| **左对齐** | 产品图在底图左侧垂直均匀排列 | 侧边栏展示 |
| **居中** | 产品图在底图中央垂直均匀排列 | 主图展示 |
| **右对齐** | 产品图在底图右侧垂直均匀排列 | 侧边栏展示 |
| **水平分布** | 产品图在底图顶部水平均匀排列 | 顶栏展示 |
| **九宫格** | 3×3 网格，产品图居中填充 | 多产品概览 |
| **2列均匀** | 2列等分，均匀填充 | 对比展示 |
| **3列均匀** | 3列等分，均匀填充 | 多产品展示 |
| **自由排列** | 保持当前画布上的位置不动 | 用户自定义 |
| **用户自定义** | 保存用户创建的自定义模板 | 重复使用 |

#### 5.4.2 模板操作流程
1. 用户在画布上手动调整好一张产品图的位置/大小/旋转
2. 点击「保存为模板」，记录当前所有产品图的相对位置参数
3. 选择「批量套模板」下拉菜单，点击某个模板名
4. 所有产品主图重新按模板布局排列到底图上
5. 用户可微调后点击「预览」或「导出」

#### 5.4.3 模板位置计算
- 以底图尺寸为参考系（`bgWidth × bgHeight`）
- 模板保存的是相对比例（`x/bgW, y/bgH, scale`）而非绝对像素
- 底图更换时，模板自动按新底图尺寸重新计算位置
- 产品图保持原始宽高比

### 5.5 预览

- **实时预览**：画布区域即实时预览，所有操作即时反映
- **全屏预览**：点击「预览」按钮，弹出模态框展示合成后的完整图片
- **对比预览**：左右分屏，左侧底图原图，右侧合成后效果图
- **缩放**：全屏预览中支持鼠标滚轮缩放查看细节

### 5.6 导出下载

#### 5.6.1 单张下载
- 选中某张产品图 → 右键菜单「单独导出」→ 仅合成该图到底图上并下载
- 或在全屏预览中点击「下载此图」

#### 5.6.2 批量下载
- 点击底部「导出 ZIP」按钮
- 将每张产品图分别与底图合成，生成多张图片
- 打包为 ZIP 下载，命名规则：`合成图_01.png`、`合成图_02.png` ...
- 可选输出格式：PNG（默认，保留透明）、JPG（可调质量 1-100）

#### 5.6.3 导出参数
| 参数 | 默认值 | 说明 |
|------|--------|------|
| 输出格式 | PNG | PNG / JPG / WebP |
| JPG 质量 | 95 | 仅 JPG 格式生效 |
| 输出尺寸 | 底图原始尺寸 | 可选：原始 / 1080px / 1500px / 自定义 |
| 文件名前缀 | `合成图` | 支持自定义 |

## 八、Canvas 合成算法

### 8.1 合成流程

```
1. 创建离屏 Canvas（尺寸 = 底图原始尺寸）
2. 绘制底图 → ctx.drawImage(bgImage, 0, 0)
3. 遍历 placements[]，对每个 visible 的产品图：
   a. ctx.save()
   b. ctx.translate(placement.x, placement.y)
   c. ctx.rotate(placement.rotation * Math.PI / 180)
   d. ctx.scale(placement.scale, placement.scale)
   e. ctx.drawImage(productImg, -imgW/2, -imgH/2)
   f. ctx.restore()
4. 返回 canvas.toBlob() 或 canvas.toDataURL()
```

### 8.2 性能优化
- 底图和产品图使用 `Image` 对象预加载，缓存到内存
- 拖拽过程中使用 requestAnimationFrame 做渲染节流
- 批量导出使用 Web Worker 并行合成（或顺序执行避免内存峰值）
- 导出 PNG 时使用 `canvas.toBlob('image/png')` 而非 toDataURL（内存更优）

## 九、UI/UX 细节

### 9.1 主题
- 支持深色/浅色主题切换（复用现有 ThemeToggle 组件模式）
- 画布区域背景：浅色用浅灰棋盘格（表示透明），深色用深灰棋盘格

### 9.2 快捷键
| 快捷键 | 功能 |
|--------|------|
| Delete / Backspace | 删除选中产品图 |
| Ctrl+D | 复制选中产品图 |
| Ctrl+Z | 撤销上一步操作 |
| Ctrl+Shift+Z | 重做 |
| Ctrl+G | 开关网格辅助线 |
| Ctrl+0 | 画布适应窗口 |
| Ctrl+1 | 画布实际尺寸 |
| Ctrl+E | 快速导出 ZIP |

### 9.3 空状态
- 底图未上传：画布区域显示虚线框 + 「上传底图开始创作」提示
- 产品图未上传：底图上方无叠加内容，侧边栏提示「上传产品主图」
- 无选中产品图：画布上不显示任何操作手柄

### 9.4 错误处理
- 文件格式不支持：Toast 提示「仅支持 JPG/PNG/WebP 格式」
- 底图上传超过 1 张：Toast 提示「底图仅支持 1 张，已替换为最新上传」
- 产品图超过 20 张：Toast 提示「建议不超过 20 张以保证性能」
