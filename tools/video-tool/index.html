如何获取视频直链？

方法一：浏览器插件（推荐）

  1. 安装「猫抓」或「Stream」浏览器插件
  2. 在微信PC端打开视频号（使用系统浏览器）
  3. 播放视频，插件会自动捕获视频URL
  4. 复制URL粘贴到上方

方法二：抓包工具

  1. 安装 Fiddler 或 Charles
  2. 配置代理抓取微信流量
  3. 播放视频，找到 mp4/m3u8 URL
  4. 复制URL粘贴到上方
